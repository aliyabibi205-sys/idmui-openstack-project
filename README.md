Identity Management User Interface system using Openstack Keystone APIs and Flask
# Identity Management User Interface (IDMUI)

This project is a web-based Identity Management System developed using OpenStack Keystone APIs.

## Features
- User Registration via Keystone APIs
- Login System with Token Generation
- Token-based Authentication
- MariaDB Database Integration

## Technologies Used
- Python (Flask)
- OpenStack Keystone
- MariaDB
- Ubuntu (Linux)
- VirtualBox

## How to Run
This project runs in a virtual environment:

1. Install Oracle VirtualBox
2. Setup Ubuntu Virtual Machine
3. Install OpenStack Keystone
4. Configure environment
5. Run Flask application


## Note
This project runs in a virtual environment (Ubuntu and OpenStack), so it cannot be deployed directly online.
